# Code Style Skill

This skill helps maintain a consistent code style across the project.

## Languages and Frameworks

This project uses multiple languages and frameworks. The code style guidelines are organized by:

- **TypeScript/JavaScript** - See [typescript.md](typescript.md)
- **Angular** - See [angular.md](angular.md)

## General Principles

- Follow the existing patterns in the codebase
- Use the provided formatters and linters
- Keep code readable and maintainable
- Write self-documenting code with clear naming
- When creating request and response objects, add them next to each other - add the response right after the request
- All UI texts must be in Russian. All comments in code in English.
